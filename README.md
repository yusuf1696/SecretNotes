# Secret Notes - Secure Text Encryptor
A basic Python application built using Tkinter that allows users to save and decrypt secret notes locally. It implements Fernet (AES-128) encryption using secure keys generated via SHA-256 hashing.

## Usage
1. Enter your secret text and a title for that.
2. Click the 'Save & Encrypt' button with your password.
3. Go to the 'secrets.txt' file and find your note's title.
4. Copy the encrypted text below title and paste it back into the text field on the main interface
5. Enter correct password and click the 'Decrypt' button to see your note.
